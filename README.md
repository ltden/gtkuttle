gtkuttle
========

A GTK sshuttle GUI with tray icon, written in Python.

Requirements
------------

* PyGTK2
* gksudo
* sshuttle in $PATH


Optional requirements
---------------------

* (x11-)ssh-askpass if you don't use keys for your ssh session
